Welcome to Dan's Budgeting Tracker!

Right now, there's not much here, but here are some goals for this application:

  • main page display main trends
  • secondary page with all transactions per category per month
  • Page to edit add, and view categories for the transactions
  • upload page either on its own or within the transactions page

Table design so far

transaction table - Date (or month and day as numbers) | name as varchar(255) | amount as float | category as varchar(50)

categories table - id as int | category as varchar(50)
